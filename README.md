# Used-Car-Price-Prediction-Using-Regression-Analysis-
This project applies Linear Regression to predict used car prices based on vehicle features such as brand, engine specifications, mileage, and other attributes.

The workflow includes:

- Data cleaning and handling missing values

- Feature engineering (dummy variable encoding)

- Log transformation to improve model assumptions

- Multicollinearity checking using VIF

- Model training and evaluation using scikit-learn

- Interpretation of regression coefficients

The goal of this project is to demonstrate a complete end-to-end regression pipeline, from raw dataset to price prediction and model diagnostics.

Technologies used:

- Python

- Pandas & NumPy

- Matplotlib & Seaborn

- Scikit-learn

- Statsmodels
